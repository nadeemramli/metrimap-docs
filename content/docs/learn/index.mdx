The [Concepts](/docs/concepts) explain *what* Canvasm is. This track teaches you how
to **think** with it — how to turn a fuzzy goal into a model you can act on and
review. No technical background needed.

## How to use this track

Read the lessons in order the first time — each builds on the last. If you learn by
doing, jump straight to the [worked example](/docs/learn/example-growth-map) and
refer back to the lessons when you get stuck.
